# YouTube Clone – Frontend (React)

## Tech Used

- React with Vite
- Tailwind CSS
- React Router
- Redux Toolkit
- Cloudinary for video uploads

## Scripts

```bash
npm run dev         # Start dev server
```

1. Sidebar overlays below 768px
2. Video cards scale depending on sidebar state
3. Mobile/Tablet/Laptop friendly layout
